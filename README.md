# 📈 Demand Forecasting & Inventory Optimization

## Overview

This project uses Facebook Prophet to forecast daily product demand and support inventory planning decisions.

Historical retail transaction data was analyzed to identify trends and seasonal patterns before building forecasting models.

---

## Dataset

- Online Retail Dataset
- Approximately 400 unique products

---

## Objective

Predict future daily demand and compare forecasting performance against a baseline model.

---

## Tech Stack

- Python
- Prophet
- Pandas
- NumPy
- Matplotlib

---

## Methodology

- Data cleaning
- Feature engineering
- Trend analysis
- Seasonality analysis
- Forecast generation
- Model evaluation

---

## Results

- Forecasted demand across ~400 products
- Improved forecasting accuracy by approximately 28%
- MAE improved from 12,404 to 8,872
